## Visão geral

Este projeto funciona como vitrine profissional para apresentar:

- serviços oferecidos;
- projetos selecionados;
- redes sociais e contato direto;
- estilo visual autoral com animações sutis e responsividade.

## Destaques

- Layout responsivo para desktop e mobile.
- Seções com efeito de entrada por scroll.
- Carrossel leve para cards de portfólio.
- Botões e elementos visuais reutilizáveis via componentes CSS.
- Organização de arquivos em src para melhor manutenção.

## Stack

- HTML5
- CSS3
- JavaScript (vanilla)
- Bootstrap Icons (CDN)
- Devicon (CDN)

## Como executar localmente

1. Clone o repositório:

```bash
git clone https://github.com/Nathan-Hergesel/Portifolio.git
```

2. Entre na pasta:

```bash
cd Portifolio
```

3. Abra o arquivo index.html no navegador

Opcional: use a extensão Live Server no VS Code para recarregamento automático.

## Customização rápida

- Cores globais: ajuste as variáveis em :root dentro de src/styles/main.css.
- Componentes de botão: edite src/styles/components.css.
- Conteúdo e seções: atualize diretamente em index.html.
- Interações e animações em JS: src/js/main.js.

## Deploy

O projeto pode ser publicado facilmente em:
