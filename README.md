# 🚀 Portfólio Pessoal - Nathan Hergesel

Este é o repositório do meu portfólio pessoal, desenvolvido para apresentar meus projetos, habilidades e experiências no mundo do desenvolvimento.

## 🖥️ Sobre o Projeto

O objetivo deste site é servir como um cartão de visitas digital, onde mostro de forma prática o que sei construir. O projeto foca em uma interface limpa, moderna e totalmente responsiva.

---

## 🛠️ Tecnologias e Ferramentas

Para o desenvolvimento deste portfólio, utilizei:

* **HTML5**: Semântica e estrutura de dados.
* **CSS3**: Estilização personalizada, animações e layout responsivo.
* **JavaScript**: Manipulação de DOM para interatividade (menus, scroll, etc).

---

## 📁 Estrutura de Arquivos

A organização do repositório está disposta da seguinte forma:

| Arquivo/Pasta | Descrição |
| :--- | :--- |
| `index.html` | Página principal do portfólio. |
| `styles.css` | Arquivo contendo toda a identidade visual (CSS). |
| `script.js` | Funções de interatividade do site. |
| `/src` | Pasta com assets (Imagens, ícones e recursos). |
| `CNAME` | Configuração de domínio personalizado. |

---

## 🚀 Como Executar

Para ver o projeto localmente:

1.  **Clone este repositório:**
    ```bash
    git clone [https://github.com/Nathan-Hergesel/Portifolio.git](https://github.com/Nathan-Hergesel/Portifolio.git)
    ```
2.  **Acesse a pasta:**
    ```bash
    cd Portifolio
    ```
3.  **Abra o arquivo no navegador:**
    Basta clicar duas vezes no `index.html` ou usar a extensão *Live Server* do VS Code.
